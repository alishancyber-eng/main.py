# MySecurityTool

Multi-threaded Python port scanner with report generation and safety checks.

## Contents
- `MySecurityTool/scanner.py` — main scanner script
- `scan_report.txt` — last generated scan report

## Quick start
1. Create a virtualenv and install Python 3.8+ (no extra packages required).
2. Run the scanner:

```powershell
python MySecurityTool\scanner.py
```

3. Follow prompts, report will be saved to `scan_report.txt`.
